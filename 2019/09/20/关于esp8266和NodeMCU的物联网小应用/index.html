关于esp8266和NodeMCU的物联网小应用

发表于 2019-09-20

最近在折腾esp8266，因为在做arduino的东西的时候发现了它，完美解决了硬件联网的问题。

ESP8266介绍

由乐鑫公司开发的一款芯片，拥有32为Tensilica处理器，低功耗，高集成的Wifi芯片，是物联网开发的必备之物。

而且，它已经移植到了arduino平台，这也是吸引我的理由，还有MicroPython平台，也可以使用Lua语言去开发，在全球都有很大的影响力。

NodeMCU介绍

NodeMCU是一个开源的物联网平台。 它使用Lua脚本语言编程。该平台基于eLua开源项目,底层使用ESP8266 sdk 0.9.5版本。该平台使用了很多开源项目, 例如 lua-cjson, spiffs. NodeMCU包含了可以运行在esp8266Wi-FiSoC芯片之上的固件,以及基于ESP-12模组的硬件。

通过网页点亮第一颗物联网LED

这里我选的的NodeMCU，它集成了esp8266，可以使用安卓Micro-USB数据线供电以及烧录程序，最重要的是，可以在Arduino IDE上开发。

配置开发环境

  1. 打开Arduino IDE，在首选行的附加开发板管理网址添加：http://arduino.esp8266.com/stable/package_esp8266com_index.json
  2. 工具——开发板——开发板管理器，输入esp8266，点击install
  3. 开发板选择NodeMCU 1.0 ESP-12E Module
  4. 选择正确的串口

编写程序

一个简单的网页，有LedOn和LedOff两个选项，对应小灯的亮和灭，程序中的SSID和password对应你的wifi名字和wifi密码，确保你的NodeMCU和你打开网页的设备在同一局域网下。

接线

这里我定义LED的引脚是D7，找到D7，和LED正极相连，然后负极串个电阻，和NodeMCU的GND相连，最后接上usb线，就可以把程序烧进去了。

点亮一个LED之后

现在通过网页上的两个按钮就可以控制LED亮灭了，每一个复杂的单片机项目都是从点亮一个LED开始的，通过NodeMCU将单片机和网络连接，就可以干很多事情了。
