LeetCodeNo.1-Two Sum

发表于 2019-08-21

两数之和，java实现。

description

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example：

方法一

双循环，暴力解决

方法二

构造一个map，第一个循环将数组存进去，第二个循环，设置一个变量存储当前数字和target之间的差值，判断map的key中是否包含这个差值。

方法三

只需要一个循环，在建立map的每一次循环的时候，同时检查是否有解。

测试

随机数组放一千万个数，然后给定一个target，明显方法三用时短些。
