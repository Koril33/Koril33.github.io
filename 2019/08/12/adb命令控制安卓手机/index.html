adb命令控制安卓手机

发表于 2019-08-12

adb就是连接安卓手机和PC端的桥梁，可以让用户在电脑上操纵手机。

环境

  1. 下载adb工具包
  2. 配置系统环境变量（cmd键入adb，出现信息，就说明配置好了）
  3. 打开手机开发者模式
  4. 把开发者模式下USB调试相关权限开起来

USB连接

在cmd中键入adb device，如果出现:

List of devices attached
一串数字加字母 device

说明连接成功了。

adb模拟按键

使用命令adb shell进入shell模式

需要退出shell模式时，使用命令exit

进入shell后，就可以用电脑来模拟手机操作了。举几个简单的例子：

点亮屏幕（按下手机电源键）

input keyevent 26

HOME键

input keyevent 3

增加音量

input keyevent 24

降低音量

input keyevent 25

点击

input tap x y

滑动

input swipe startX startY endX endY ms(最后一个参数是总体时间)

输入文字

input text 内容

所以，只要知道keyevent对应的数字编号就可以做出相应的命令，tap，swipe都是很有用的操作，可以自己写出小脚本来控制手机，关于手机上的坐标问题，我用的小米是可以在开发者选项中打开指针位置的，这个调试功能很方便。

adb打开摄像头，截图，传图

打开摄像头：

adb shell am start -n com.android.camera/com.android.camera.Camera

截图（保存到sdcard/路径下）

adb shell screencap -p /sdcard/01.png

将文件发送到电脑

比如。我想把刚刚的01.png发给电脑，输入命令：adb pull sdcard/01.png C:\Users\dingj\Desktop\photo

具体命令格式是：adb pull <手机文件路径> <电脑文件路径>

电脑文件传给手机

又比如，我电脑上对刚刚传过来的图片进行了处理，想传回去，输入命令：adb push C:\Users\dingj\Desktop\photo\02.png sdcard

这样，手机sdcard路径下应该多了一张02.png

具体命令格式：adb push <电脑文件路径> <手机文件路径>
