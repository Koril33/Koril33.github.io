'Opencv+Python图像处理入门'

发表于 2019-08-01

因为比赛的需要，要求识别颜色、形状，所以趁有空把Opencv+Python学起来。

使用的库：numpy, opencv

在同级文件下，放个图片cat.jpg作为测试图片。

读取图片、显示图片

  1. cv2.imread(文件名，标记)：用来读取图片，标记可选参数如下：
    • cv2.IMREAD_COLOR：读取彩色模式
    • cv2.IMREAD_GRAYSCALE：读取灰度模式
  2. cv2.imshow(窗口名，图片)：用来显示图片
  3. cv2.imwrite(文件名，图片)：用来保存图片
  4. cv2.waitKey(delay)等待键盘输入，delay默认为0
  5. cv2.destroyAllWindows()删除窗口
