Linux基础命令（一）

发表于 2019-08-15

关于Linux命令，ls，cd，pwd，mkdir，rmdir，cp，mv，rm一些简单的参数和操作。

ls

Linux的ls命令用于显示目录内容。

参数：

  • -a 显示所有文件及目录，包括“.”（当前目录）和“..”（父目录）
  • -A 同a，但不包括“.”和“..”
  • -r 将文件以反次序显示（原定依字母次序排列）
  • -l 列出详细信息，类型，权限，拥有者，大小等等
  • -F 列出的文件后加符号，如执行文件加“*”，目录加“/”
  • -R 若目录下有文件，则将之下的文件依次列出

cd

Linux的cd命令用于切换目录。

参数：

  • ~ 进入home目录
  • / 进入根目录
  • .. 进入父目录（一直使用“..”最后会进入根目录）
  • -是返回刚才的目录
  • ../.. 跳到目录上两层

pwd

Linux的pwd命令用于查看当前所在目录。

mkdir

Linux的mkdir命令用于创建目录。

小栗子：

  1. 想要在在工作目录下，创建一个名为test1的子目录

    mkdir test1

  2. 想要在test2目录中，创建一个名为test的文件，若test2原先不存在，则必须加上参数“-p”，否则会报如下错误：”mkdir: cannot create directory ‘test2/test’: No such file or directory”。

    mkdir -p test2/test

rmdir

Linux的mkdir命令用于删除空目录。

要注意的是，删除的目录必须为空。

小栗子：

  1. 在text1/text/目录下新建一个111.txt文件，也就是让目录不为空，这时候执行rmdir test1，汇报如下错误：rmdir: failed to remove ‘test1’: Directory not empty。

  2. 现在把111.txt先rm掉，这时候text就成了空目录，可以依然不能rmdir掉text1，因为它还包含了text目录，但可以空目录text就可以使用rmdir了。

    rmdir test1/test

  3. 如果我想要删掉test的同时，检查其父目录text1是否为空，若为空，则一并删除。添加参数“-p”就可以了。

    rmdir -p test1/test

cp

Linux的cp命令用于复制文件或目录。

参数：

  • -r 若复制的文件是个目录，将该目录下的所有子目录，和文件都复制到指定路径中

  • -R 和命令“-r”类似，但在有些情况不同

  • -p 除复制文件内容外，还把修改时间和访问权限也复制到新文件中

  • -f 复制到的指定目录存在，同名文件时，进行覆盖，不提醒用户。

    这里有个坑，我发现加了“-f”参数，系统还是会让我手动确认，其实跟系统设置有关，cp这个命令被系统设置了别名，相当于“cp -i”，也就是说，在执行cp的时候，其实实在执行cp -if，解决办法就是使用原生的cp，可以执行命令/bin/cp -f,就可以了。

  • -i 与-f相反，在覆盖前会给出提示，要求用户确认

  • -l 不复制文件，只是生成链接文件

  • -a 通常在复制目录的时候使用，相当于命令“-dpR”

mv

Linux的mv命令用于为文件或目录重命名，以及将文件或目录移入其他位置。

参数：

  • -i 若指定目录已有同名文件，则提示用户
  • -f 覆盖时不进行提示

小栗子：

  1. 将文件或目录test1更名为test2

    mv test1 test2

  2. 将test1放入mytest/test2目录中，若原来没有test2这个目录，就将test1移入mytest目录下，并将其更名为test2

    mv test1 mytest/test2

  3. 将mytest/test2下的所有文件和目录移到当前目录下

    mv mytest/test2/* ./

rm

参数：

  • -i 删除前提醒用户
  • -f 不提醒直接删
  • -r 将目录及以下的所有文件删除

小栗子：

  1. 删除文件hello.txt

    rm hello.txt

  2. 删除目录，单独使用命令“rm”会报错：rm: cannot remove ‘test1’: Is a directory

  3. 删除指定目录及其以下所有文件

    rm -r test1

  4. 删除当前目录下的所有文件

    rm -rf ./*或者rm -rf *

文件一旦通过命令“rm”删除，无法恢复，所以要格外小心！
