头插法和尾插法以及打印链表

  • 头插法：在head节点后面插入新节点。
  • 尾插法：在tail节点后面插入新节点。
  • 插入1，2，3：头插法的输出是0-3-2-1，也就是最新的节点跟在head后面，尾插法的输出是0-1-2-3，最新的节点放在最末尾。（假设head节点存放数字0，这样会看的清楚一些）。
